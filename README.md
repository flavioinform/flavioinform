<h2>Proyectos</h2>

🚗 <b>ECOMONITOR</b>  
Vehículo robótico autónomo para **monitoreo de gases industriales** utilizando sensores (CO, H₂S, CH₄), comunicación **LoRa**, GPS y visualización en aplicación móvil.

📡 <b>Controlador Robótico LoRa</b>  
Sistema de control remoto para vehículos robóticos usando **ESP32 LoRa, joysticks y comunicación de largo alcance**.

🛰️ <b>Sistemas de Monitoreo IoT</b>  
Desarrollo de redes de sensores con **ESP32 y LoRa para monitoreo ambiental y telemetría**.
